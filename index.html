Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

HTML 5 (Hyper Text Languaje, version 5) es la quinta revision importante del lenguaje básico de la World Wide Web. HTML.

Usa la extensión .html

  • CSS 3

Hoja de estilo en cascada o CSS (siglas en ingles de cascading style sheets) es un lenguaje usado por definir y drear la presentación de un documento estructurado escrito en HTML.

Usa la extención css

  • JavaScript

JavaScript (abreviado comúnmente "JS") es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementando como parte de un navegador web permitiendo mejoras en la interfaz de usuario y página web dinámicas.

Usa la extención .js
